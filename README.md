# Vue 3 + TypeScript + Vite

## 推荐开发环境

- `vscode` + `eslint 插件` + `stylelint 插件` + `Volar 插件`

## 安装

- `npm install`

## 开发

- `npm run style:dts`: 生成`*.module.scss`的类型定义文件，这样import scss 的时候有类型提示。
- `npm run dev`: 启动 vite 开发服务器。

## 发布

- `npm run build`

## 配置文件

- `envs/*`
- `npm run env`: 自动更新 `env.d.ts`
